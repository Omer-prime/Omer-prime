## Selected Work

### [RouteLog Pro HOS Planner](https://github.com/Omer-prime/routelog-pro-hos-planner)
A full-stack HOS trip planner for truck drivers with route mapping, duty-status timeline, fuel/rest stop planning, 70-hour cycle tracking, and printable FMCSA-style daily logs.

### [AI Dev Guardian Core](https://github.com/Omer-prime/AI-Dev-Guardian-Core)
An open-source local SAST scanner that finds risky code patterns, generates actionable fixes, and exports SARIF reports for secure development workflows and CI pipelines.

### [Gold Streaming App](https://github.com/Omer-prime/gold-streaming-app)
A Poppo-style live streaming platform with real-time chat, gifts, coins wallet, LiveKit rooms, admin dashboard, Next.js web app, and React Native mobile integration.

### [Pavement SystemX MVP](https://github.com/Omer-prime/pavement-systemx-mvp)
A road inspection and pavement analysis MVP focused on route scanning, visual mapping, AI-assisted estimates, and professional infrastructure workflow planning.

### [Our Common Commute](https://github.com/Omer-prime/our-common-commute)
A monorepo full-stack ride booking web app with Next.js, Node.js REST API, trip status flow, Tailwind UI, and Google Maps based route experience.

### [ESU](https://github.com/Omer-prime/esu)
A bridge service and workflow layer for integrating app onboarding with WhatsApp Business APIs, step-based automation, status syncing, and messaging flows.

### Medical Companion
A safety and healthcare-focused companion application designed to support check-ins, monitoring workflows, caregiver connections, and user-friendly medical support experiences.

### BrightPath
An AI-powered learning platform for children with learning difficulties, focused on parent/teacher flows, OTP onboarding, language selection, dashboards, and child-friendly learning experiences.

### Pak Test Maker
A React Native and TypeScript mobile app that helps users generate custom class tests in PDF format based on selected subjects, chapters, and test preferences.
